灵犀智空 Lingxi BL 感知模组 — 固件工程

工程概述

VSCode + Linux 环境下的跨平台固件工程，支持 STM32N6（主控）和 ESP32-C6（通信）。

目录结构

lingxi-bl-firmware/
├── .vscode/              # VSCode配置
├── stm32/                # STM32N657主控固件
│   ├── Core/             # 核心代码
│   ├── Drivers/          # HAL驱动
│   ├── Middlewares/      # FreeRTOS等中间件
│   ├── App/              # 应用层
│   └── CMakeLists.txt    # CMake构建
├── esp32/                # ESP32-C6通信固件
│   ├── main/             # 主程序
│   ├── components/       # 组件
│   └── CMakeLists.txt    # CMake构建
├── shared/               # 共享代码
│   ├── protocol/         # 通信协议
│   └── utils/            # 工具函数
├── tests/                # 单元测试
├── tools/                # 开发工具
└── docs/                 # 文档

快速开始

环境要求

  • VSCode + CMake Tools插件
  • arm-none-eabi-gcc (STM32)
  • ESP-IDF v5.2+ (ESP32-C6)
  • OpenOCD (调试)

STM32构建

cd stm32
cmake -B build -DCMAKE_TOOLCHAIN_FILE=cmake/arm-none-eabi-gcc.cmake -G Ninja
cmake --build build

ESP32构建

cd esp32
idf.py build
idf.py flash

系统架构

  • STM32N657: 主控 + Neural-ART NPU（目标检测/避障/跟踪）
  • ESP32-C6: SDIO Slave + WiFi 6 + BLE 5 + Zigbee
  • 通信协议: 自定义轻量协议 over SDIO
